# <a name="get-started-with-hello-box-connector"></a>Iniziare con il connettore di casella hello
Connettersi tooBox e creare i file, eliminare i file e molto altro. Con Box è possibile:

* Compilare il flusso di business in base ai dati hello che si ottiene dalla casella. 
* Usare i trigger quando un file viene creato o aggiornato.
* Usare le azioni per copiare un file, eliminarlo e così via. Tali azioni ottengono una risposta e quindi apportare output di hello per le altre azioni. Ad esempio, quando un file viene modificato in Box, è possibile usare il file e la posta elettronica tramite Office 365.

Per iniziare subito a creare un'app per la logica, vedere [Creare un'app per la logica](../logic-apps/logic-apps-create-a-logic-app.md).
