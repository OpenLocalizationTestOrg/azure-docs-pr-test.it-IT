## <a name="data-asset"></a><span data-ttu-id="ae333-110">Asset di dati</span><span class="sxs-lookup"><span data-stu-id="ae333-110">Data asset</span></span>

## <a name="data-asset-location"></a><span data-ttu-id="ae333-113">Posizione degli asset di dati</span><span class="sxs-lookup"><span data-stu-id="ae333-113">Data asset location</span></span>

## <a name="structural-metadata"></a><span data-ttu-id="ae333-117">Metadati strutturali</span><span class="sxs-lookup"><span data-stu-id="ae333-117">Structural metadata</span></span>

## <a name="descriptive-metadata"></a><span data-ttu-id="ae333-121">Metadati descrittivi</span><span class="sxs-lookup"><span data-stu-id="ae333-121">Descriptive metadata</span></span>

## <a name="preview"></a><span data-ttu-id="ae333-132">Preview</span><span class="sxs-lookup"><span data-stu-id="ae333-132">Preview</span></span>
<span data-ttu-id="ae333-133">L'anteprima nel Catalogo dati di Azure è uno snapshot di 20 record al massimo che possono essere estratti dall'origine dati durante la registrazione e archiviati nel catalogo con i metadati dell’asset di dati.</span><span class="sxs-lookup"><span data-stu-id="ae333-133">A preview in Azure Data Catalog is a snapshot of up to 20 records that can be extracted from the data source during registration, and stored in the catalog with the data asset metadata.</span></span> <span data-ttu-id="ae333-134">L'anteprima può aiutare gli utenti a individuare un asset di dati per comprenderne meglio lo scopo e la funzione.</span><span class="sxs-lookup"><span data-stu-id="ae333-134">The preview can help users who discover a data asset better understand its function and purpose.</span></span> <span data-ttu-id="ae333-135">In altre parole, i dati di esempio possono essere più importanti dei soli nomi di colonna e tipi di dati.</span><span class="sxs-lookup"><span data-stu-id="ae333-135">In other words, seeing sample data can be more valuable than seeing just the column names and data types.</span></span>

<span data-ttu-id="ae333-138">Un profilo dati nel Catalogo dati di Azure è uno snapshot di metadati a livello di tabella e a livello di colonna su un asset di dati registrati che può essere estratto dall'origine dati durante la registrazione e archiviato nel catalogo con i metadati dell’asset di dati.</span><span class="sxs-lookup"><span data-stu-id="ae333-138">A data profile in Azure Data Catalog is a snapshot of table-level and column-level metadata about a registered data asset that can be extracted from the data source during registration, and stored in the catalog with the data asset metadata.</span></span> <span data-ttu-id="ae333-139">Il profilo dati può aiutare gli utenti a individuare un asset di dati per comprenderne meglio lo scopo e la funzione.</span><span class="sxs-lookup"><span data-stu-id="ae333-139">The data profile can help users who discover a data asset better understand its function and purpose.</span></span> <span data-ttu-id="ae333-140">Analogamente alle anteprime, i profili dati devono essere selezionati esplicitamente dall'utente durante la registrazione.</span><span class="sxs-lookup"><span data-stu-id="ae333-140">Similar to previews, data profiles must be explicitly selected by the user during registration.</span></span>
