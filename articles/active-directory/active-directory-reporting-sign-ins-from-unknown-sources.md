# <a name="sign-ins-from-unknown-sources"></a>Accessi da origini sconosciute
Questo rapporto indica gli utenti che hanno effettuato correttamente il tooyour directory mentre è assegnato un indirizzo IP che è stata riconosciuta da Microsoft come indirizzo IP proxy anonimo (ad esempio, un indirizzo IP Tor) client. Questi proxy vengono spesso utilizzati dagli utenti che desiderano toohide indirizzo IP del computer e possono essere utilizzate per fini dannosi.

Risultati del report verranno mostrati il numero di hello di volte in cui un utente è connesso correttamente nella directory tooyour da tale indirizzo e l'indirizzo IP del proxy hello.
