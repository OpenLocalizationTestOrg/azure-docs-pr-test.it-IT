# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Linee guida dell'utilità Caricamento file per le nuove richieste di supporto di Microsoft Azure
Quando si [creare una richiesta di supporto](https://portal.azure.com/#create/Microsoft.Support), utilità di caricamento File hello consentono facilmente caricare i file di diagnostica o qualsiasi altro file che si ritiene siano rilevanti toohello incidente.  

> [!NOTE]
> tooprotect la privacy degli utenti, non includere informazioni personali nel caricamento.
>
>

Quando si utilizza l'utilità di caricamento File hello, attenersi alle seguenti indicazioni:

* Tutti i file devono avere un'estensione di file, ad esempio docx o xlsx. Vedere la tabella hello per elenco hello di estensioni di file consentiti.
* nome del file Hello deve essere non più di 110 caratteri.
* Non è possibile caricare più di un file.
* Le dimensioni del file non devono superare i 4 MB.
