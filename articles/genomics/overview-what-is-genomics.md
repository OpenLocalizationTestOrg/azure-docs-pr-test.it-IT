# <a name="what-is-microsoft-genomics"></a>Cos'è Genomica di Microsoft?
Genomics Microsoft offre un'implementazione di cloud di Aligner il Wheeler Burrows (BWA) e il Toolkit di analisi genoma (GATK) per l'analisi secondario. Il servizio certificato ISO e conforme alle normative HIPAA e offre prezzi prevedibilità alle proprie esigenze di sequenziazione genoma. Informazioni su come usare il servizio Microsoft Genomics e integrazione con l'API leggendo la documentazione, esercitazioni ed Guide rapide.

## <a name="support-your-most-demanding-sequencing-needs"></a>Supporta i carichi di lavoro di sequenziazione più complessi
Anziché gestire data center di proprietà, approfitta dell'esperienza globale di Microsoft nell'esecuzione di carichi di lavoro a livello di esabyte. Il nostro implentation cloud del BWA-GATK è altamente concordant con pipeline consigliate migliore dell'Institute ampia.


## <a name="keep-your-business-running"></a>Le operazioni in esecuzione
Genomics Microsoft offre un disponibilità del 99,99% contratto di servizio (SLA) per la ricezione di richieste di flusso di lavoro. Altre informazioni sul Genomics Microsoft [contratto di servizio](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Protezione dei dati
Il servizio Microsoft Genomics è ISO 27001, ISO 27018 e ISO 9001 certificata e conforme alle normative HIPAA. Visitare il [Centro protezione di Azure](https://www.microsoft.com/TrustCenter/Security/default.aspx) per informazioni sulla sicurezza della piattaforma Azure.
