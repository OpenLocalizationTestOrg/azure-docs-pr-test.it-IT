<span data-ttu-id="09342-127">Un vantaggio dell'uso del portale consiste nel fatto che Ricerca di Azure può solitamente generare automaticamente uno schema di indice predefinito leggendo i metadati del set di dati di origine.</span><span class="sxs-lookup"><span data-stu-id="09342-127">An advantage to using the portal is that Azure Search can usually generate a default index schema for you by reading the metadata of the source dataset.</span></span> <span data-ttu-id="09342-128">È possibile modificare l'indice generato fino al termine dell'elaborazione dell'indice.
