## <a name="iot-hardware-manufacturerintegrator"></a><span data-ttu-id="012c4-119">Produttore/integratore di hardware IoT</span><span class="sxs-lookup"><span data-stu-id="012c4-119">IoT hardware manufacturer/integrator</span></span>

* <span data-ttu-id="012c4-121">**Impostare l'hardware sui requisiti minimi**: la progettazione dell'hardware deve includere esclusivamente le funzionalità minime necessarie per il funzionamento.</span><span class="sxs-lookup"><span data-stu-id="012c4-121">**Scope hardware to minimum requirements**: The hardware design should include the minimum features required for operation of the hardware, and nothing more.</span></span> <span data-ttu-id="012c4-122">Ad esempio, è possibile includere porte USB solo se necessario al funzionamento del dispositivo.</span><span class="sxs-lookup"><span data-stu-id="012c4-122">An example is to include USB ports only if necessary for the operation of the device.</span></span> <span data-ttu-id="012c4-123">Queste funzionalità aggiuntive espongono il dispositivo a vettori di attacchi indesiderati, che sarebbe opportuno evitare.</span><span class="sxs-lookup"><span data-stu-id="012c4-123">These additional features open the device for unwanted attack vectors that should be avoided.</span></span>
* <span data-ttu-id="012c4-124">**Realizzare hardware a prova di manomissione**: meccanismi integrati per il rilevamento di manomissioni fisiche, ad esempio l'apertura del coperchio del dispositivo o la rimozione di una parte del dispositivo.</span><span class="sxs-lookup"><span data-stu-id="012c4-124">**Make hardware tamper proof**: Build in mechanisms to detect physical tampering, such as opening of the device cover or removing a part of the device.</span></span> <span data-ttu-id="012c4-125">Questi segnali di manomissione possono essere inseriti nel flusso dei dati caricati nel cloud, al fine di segnalare tali eventi agli operatori.</span><span class="sxs-lookup"><span data-stu-id="012c4-125">These tamper signals may be part of the data stream uploaded to the cloud, which could alert operators of these events.</span></span>
