# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Riscatto dell'invito di Collaborazione B2B di Azure Active Directory

## <a name="azure-ad-and-microsoft-account-users"></a>Utenti con account Azure AD e account Microsoft
Per gli utenti con account di Azure AD esistente o un account Microsoft, l'esperienza di riscatto di hello è facile quanto l'accesso.

## <a name="social-id-user-first-time-redemption"></a>Primo riscatto dell'ID utente per social network
Collaborazione AD B2B Azure rende più semplice per qualsiasi toobe indirizzo di posta elettronica utilizzato di riscatto. Esaminare esperienza riscatto hello quando viene usato un indirizzo di posta elettronica non Microsoft per la collaborazione B2B.
