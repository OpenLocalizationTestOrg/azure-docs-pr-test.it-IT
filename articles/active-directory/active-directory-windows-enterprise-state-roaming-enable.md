## <a name="data-retention"></a><span data-ttu-id="f38a1-132">Conservazione dei dati</span><span class="sxs-lookup"><span data-stu-id="f38a1-132">Data retention</span></span>
<span data-ttu-id="f38a1-133">I dati sincronizzati in Azure tramite il servizio Enterprise State Roaming verranno mantenuti per un periodo illimitato, a meno che non venga eseguita un'operazione di eliminazione manuale o che si determini che i dati specifici sono obsoleti.</span><span class="sxs-lookup"><span data-stu-id="f38a1-133">Data synced to Azure via Enterprise State Roaming will be retained indefinitely unless a manual delete operation is performed or the data in question is determined to be stale.</span></span> 

<span data-ttu-id="f38a1-134">**Eliminazione esplicita:** i dati vengono eliminati quando un amministratore di Azure elimina un utente o una directory oppure quando un amministratore richiede in modo esplicito l'eliminazione dei dati.</span><span class="sxs-lookup"><span data-stu-id="f38a1-134">**Explicit deletion:** The data is deleted when an Azure admin deletes a user or a directory or an admin requests explicitly that data is to be deleted.</span></span>

* <span data-ttu-id="f38a1-135">**Eliminazione dell'utente**: quando un utente viene eliminato in Azure AD, i dati mobili dell'account utente saranno contrassegnati per l'eliminazione ed eliminati entro 90-180 giorni.</span><span class="sxs-lookup"><span data-stu-id="f38a1-135">**User deletion**: When a user is deleted in Azure AD, the user account roaming data will be marked for deletion and will be deleted between 90 to 180 days.</span></span> 
* <span data-ttu-id="f38a1-136">**Eliminazione della directory**: l'eliminazione di un'intera directory in Azure AD è un'operazione immediata.</span><span class="sxs-lookup"><span data-stu-id="f38a1-136">**Directory deletion**: Deleting an entire directory in Azure AD is an immediate operation.</span></span> <span data-ttu-id="f38a1-137">Tutti i dati delle impostazioni associati alla directory saranno contrassegnati per l'eliminazione ed eliminati entro 90-180 giorni.</span><span class="sxs-lookup"><span data-stu-id="f38a1-137">All the settings data associated with that directory will be marked for deletion and will be deleted between 90 to 180 days.</span></span> 
* <span data-ttu-id="f38a1-138">**Eliminazione su richiesta**: se l'amministratore di Azure AD vuole eliminare manualmente i dati delle impostazioni o di un utente specifico, può inviare un ticket al [supporto tecnico Azure](https://azure.microsoft.com/support/).</span><span class="sxs-lookup"><span data-stu-id="f38a1-138">**On request deletion**: If the Azure AD admin wants to manually delete a specific user’s data or settings data, the admin can file a ticket with [Azure support](https://azure.microsoft.com/support/).</span></span> 

<span data-ttu-id="f38a1-139">**Eliminazione dei dati obsoleti**: i dati a cui non è stato eseguito l'accesso per un anno ("periodo di conservazione") verranno considerati obsoleti e potranno essere eliminati da Azure.</span><span class="sxs-lookup"><span data-stu-id="f38a1-139">**Stale data deletion**: Data that has not been accessed for one year (“the retention period”) will be treated as stale and may be deleted from Azure.</span></span> <span data-ttu-id="f38a1-140">Il periodo di conservazione è soggetto a modifiche ma non sarà inferiore a 90 giorni.</span><span class="sxs-lookup"><span data-stu-id="f38a1-140">The retention period is subject to change but will not be less than 90 days.</span></span> <span data-ttu-id="f38a1-141">I dati obsoleti possono essere un set specifico di impostazioni di Windows/dell'applicazione o tutte le impostazioni per un utente.</span><span class="sxs-lookup"><span data-stu-id="f38a1-141">The stale data may be a specific set of Windows/application settings or all settings for a user.</span></span> <span data-ttu-id="f38a1-142">ad esempio:</span><span class="sxs-lookup"><span data-stu-id="f38a1-142">For example:</span></span>

* <span data-ttu-id="f38a1-143">Se nessun dispositivo accede a una raccolta specifica di impostazioni, ad esempio se un'applicazione viene rimossa dal dispositivo o un gruppo di impostazioni come "Tema" viene disabilitato per tutti i dispositivi di un utente, la raccolta diventerà obsoleta dopo il periodo di conservazione e potrà essere eliminata.</span><span class="sxs-lookup"><span data-stu-id="f38a1-143">If no devices access a particular settings collection (e.g., an application is removed from the device, or a settings group such as “Theme” is disabled for all of a user’s devices), then that collection will become stale after the retention period and may be deleted.</span></span> 
* <span data-ttu-id="f38a1-144">Se un utente ha disattivato la sincronizzazione delle impostazioni in tutti i propri dispositivi, non verrà effettuato l'accesso ad alcun dato delle impostazioni e tutti i dati delle impostazioni per tale utente diventeranno obsoleti e potranno essere eliminati allo scadere del periodo di conservazione.</span><span class="sxs-lookup"><span data-stu-id="f38a1-144">If a user has turned off settings sync on all his/her devices, then none of the settings data will be accessed, and all the settings data for that user will become stale and may be deleted after the retention period.</span></span> 
* <span data-ttu-id="f38a1-145">Se l'amministratore della directory di Azure AD disattiva il servizio Enterprise State Roaming per l'intera directory, tutti gli utenti in tale directory non sincronizzeranno più le impostazioni e tutti i dati delle impostazioni per tutti gli utenti diventeranno obsoleti e potranno essere eliminati allo scadere del periodo di conservazione.</span><span class="sxs-lookup"><span data-stu-id="f38a1-145">If the Azure AD directory admin turns off Enterprise State Roaming for the entire directory, then all users in that directory will stop syncing settings, and all settings data for all users will become stale and may be deleted after the retention period.</span></span> 

<span data-ttu-id="f38a1-146">**Recupero dei dati eliminati**: il criterio relativo alla conservazione dei dati non può essere configurato.</span><span class="sxs-lookup"><span data-stu-id="f38a1-146">**Deleted data recovery**: The data retention policy is not configurable.</span></span> <span data-ttu-id="f38a1-147">Dopo l'eliminazione definitiva, non sarà possibile recuperare i dati.</span><span class="sxs-lookup"><span data-stu-id="f38a1-147">Once the data has been permanently deleted, it will not be recoverable.</span></span> <span data-ttu-id="f38a1-148">È tuttavia importante notare che i dati delle impostazioni verranno eliminati solo da Azure, non dal dispositivo dell'utente finale.</span><span class="sxs-lookup"><span data-stu-id="f38a1-148">However, it’s important to note that the settings data will only be deleted from Azure, not the end-user device.</span></span> <span data-ttu-id="f38a1-149">Se un dispositivo si riconnette successivamente al servizio Enterprise State Roaming, le impostazioni verranno sincronizzate e archiviate di nuovo in Azure.</span><span class="sxs-lookup"><span data-stu-id="f38a1-149">If any device later reconnects to the Enterprise State Roaming service, the settings will again be synced and stored in Azure.</span></span>
