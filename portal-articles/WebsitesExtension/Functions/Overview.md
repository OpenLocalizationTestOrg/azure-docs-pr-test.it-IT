# <a name="azure-functions-documentation"></a>Documentazione di Funzioni di Azure

Funzioni di Azure è un servizio di calcolo senza server che consente di eseguire codice su richiesta senza dover gestire l'infrastruttura o effettuare il provisioning in modo esplicito. Usare Funzioni di Azure per eseguire uno script o una porzione di codice in risposta a diversi eventi. Informazioni su come usare Funzioni di Azure con le guide rapide, le esercitazioni e gli esempi.
