# <a name="web-app-on-linux-documentation"></a>Documentazione su App Web in Linux
Web App in Linux è attualmente in anteprima pubblica e consente ai clienti toorun le app web in modo nativo in una piattaforma Linux. Questo consente una migliore compatibilità delle applicazioni per determinati tipi di applicazioni e rende più semplice toomigrate esistente le app web ospitate in una piattaforma Linux altrove in servizi di App di Azure.

I clienti possono creare app Web in Linux solo in un piano di servizio app abilitato per Linux e questo piano di servizio app può ospitare solo app Web basate su Linux. 

> Anteprima pubblica prezzi per l'App Web in Linux è il 50% di prezzo normale hello e viene calcolato il prezzo di hello dimezzamento numero hello di ore di calcolo utilizzato dal piano del servizio App.
> 
> Il periodo di anteprima pubblica di hello non viene proposto alcun contratto di servizio.
