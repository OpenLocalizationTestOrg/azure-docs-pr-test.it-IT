# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Abilitare l'esperienza di anteprima per amministrare il dominio gestito dal portale di Azure

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Il dominio gestito esistente non viene visualizzato nell'esperienza (anteprima) del portale di Azure
Si dispone di un dominio gestito di [Azure AD Domain Services](active-directory-ds-overview.md) esistente che è stato abilitato tramite il portale di Azure classico. Quando si passa al portale di Azure, nel pannello Azure AD Domain Services (anteprima) non è elencato questo dominio gestito esistente.

![Il dominio gestito esistente non viene visualizzato nel portale di Azure](./media/getting-started/missing-domain-services-instance.png)

Questo avviene perché non è stata eseguita la migrazione del dominio gestito esistente nel nuovo portale di Azure. È possibile contattare Microsoft per abilitare la nuova esperienza del portale di Azure (anteprima) per il dominio gestito esistente.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Aggiungere l'anteprima per l'esperienza del portale di Azure per Azure Active Directory Domain Services
Seguire le istruzioni nella pagina [Contatti](active-directory-ds-contact-us.md) e inviare un'e-mail. Saremo lieti di abilitare l'anteprima del portale di Azure per il dominio gestito esistente.

> [!NOTE]
> Quando l'esperienza del nuovo portale di Azure per Azure AD Domain Services è disponibile a livello generale (GA), verrà eseguita automaticamente la migrazione di tutti i domini gestiti esistenti. A questo punto, sarà possibile gestire il dominio gestito esistente dal portale di Azure.
