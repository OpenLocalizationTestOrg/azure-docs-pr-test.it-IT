# <a name="azure-database-for-postgresql-documentation"></a>Documentazione del database di Azure per PostgreSQL

Il database di Azure per PostgreSQL è un servizio di database relazionale basato sul motore di database Postgres open source. Si tratta di un database distribuito come servizio completamente gestito capace di gestire carichi di lavoro mission-critical con prestazioni prevedibili, sicurezza, disponibilità elevata e scalabilità dinamica.  È possibile sviluppare applicazioni con il database di Azure per PostgreSQL sfruttando la piattaforma e gli strumenti open source preferiti.  Informazioni su come usare il database di Azure per PostgreSQL con le guide rapide, le esercitazioni e gli esempi.
