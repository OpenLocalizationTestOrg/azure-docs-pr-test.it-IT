<span data-ttu-id="e3d94-109">Un data warehouse è una raccolta di dati caricata da una o più origini dati e usata per eseguire attività di business intelligence, quali la creazione di report e l'analisi dei dati.</span><span class="sxs-lookup"><span data-stu-id="e3d94-109">A data warehouse is a collection of data that is loaded from one or more data sources and is used to perform business intelligence tasks such as reporting and data analysis.</span></span>
