# <a name="sample-projects-for-azure-managed-applications"></a>Progetti di esempio per applicazioni gestite di Azure

La tabella seguente include collegamenti ad applicazioni gestite di Azure di esempio in GitHub.

|  |  |
| --- | --- |
| **esempi** | |
| [Applicazione (versione di valutazione o produzione) gestita in una rete virtuale nuova o esistente](https://github.com/Azure/azure-managedapp-samples/tree/master/samples/201-managed-app-using-existing-vnet) | Viene illustrato come creare le opzioni di distribuzione flessibili per i clienti. Questa applicazione gestita può essere distribuita a una nuova rete virtuale o una rete virtuale esistente. I clienti possono specificare una versione di valutazione o di produzione delle applicazioni gestite. |
| [Managed Azure Storage Account](https://github.com/Azure/azure-managedapp-samples/tree/master/samples/201-managed-storage-account) (Account di archiviazione di Azure gestito) | Distribuisce un singolo account di archiviazione. Usare questo progetto di esempio come introduzione alla creazione di applicazioni gestite. |
| [Managed Service Fabric with Azure management services](https://github.com/Azure/azure-managedapp-samples/tree/master/samples/201-managed-service-fabric) (Service Fabric con servizi di gestione di Azure gestito) | Distribuisce un cluster di Service Fabric e set di scalabilità di macchine virtuali. Include gli account di archiviazione per la registrazione e la diagnostica. |
| [Managed Web Application (IaaS) with Azure management services](https://github.com/Azure/azure-managedapp-samples/tree/master/samples/201-managed-web-app) (Applicazione Web (IaaS) con servizi di gestione di Azure gestita) | Distribuisce una macchina virtuale che ospita un'applicazione Web. |
| [Managed Web Application (IaaS) with Azure management services and Key Vault](https://github.com/Azure/azure-managedapp-samples/tree/master/samples/201-managed-web-app-using-keyvault) (Applicazione Web (IaaS) con servizi di gestione di Azure e Key Vault gestita) | Distribuisce una macchina virtuale che ospita un'applicazione Web. Fa riferimento a un insieme di credenziali delle chiavi per la password dell'amministratore. |
