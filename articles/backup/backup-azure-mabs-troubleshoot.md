# <a name="troubleshoot-azure-backup-server"></a>Risolvere i problemi del server di Backup di Azure

Utilizzare le informazioni nelle tabelle seguenti per risolvere gli errori che si verificano durante l'utilizzo di Server di Backup di Azure.

## <a name="invalid-vault-credentials-provided"></a>Sono state specificate credenziali dell'insieme di credenziali non valide 

Per risolvere questo problema, seguire [passaggi di risoluzione dei problemi](https://docs.microsoft.com/azure/backup/backup-azure-mabs-troubleshoot#registration-and-agent-related-issues).

## <a name="the-agent-operation-failed-because-of-a-communication-error-with-the-dpm-agent-coordinator-service-on-the-server"></a>Operazione dell'agente non riuscita a causa di un errore di comunicazione con il servizio Coordinatore agenti Data Protection Manager nel server 

Per risolvere questo problema, seguire [passaggi di risoluzione dei problemi](https://docs.microsoft.com/azure/backup/backup-azure-mabs-troubleshoot#registration-and-agent-related-issues). 

## <a name="setup-could-not-update-registry-metadata"></a>Non è stato possibile aggiornare i metadati del Registro di sistema

Per risolvere questo problema, seguire [passaggi di risoluzione dei problemi](https://docs.microsoft.com/azure/backup/backup-azure-mabs-troubleshoot#installation-issues).
