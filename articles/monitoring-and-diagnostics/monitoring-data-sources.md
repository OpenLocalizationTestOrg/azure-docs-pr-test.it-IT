# <a name="consume-monitoring-data-from-azure"></a>Uso di dati di monitoraggio in Azure

In hello Azure piattaforma, viene portata insieme ai dati in un'unica posizione con pipeline di monitoraggio di Azure hello di monitoraggio, ma praticamente riconosce che attualmente non tutti i dati di monitoraggio è ancora disponibile nella pipeline. In questo articolo si riepilogano hello modi in cui è possibile accedere a livello di programmazione dati di monitoraggio da servizi di Azure.
