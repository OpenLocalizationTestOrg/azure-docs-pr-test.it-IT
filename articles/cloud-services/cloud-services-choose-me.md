<span data-ttu-id="92cac-144">Servizi cloud offre inoltre funzionalità di monitoraggio.</span><span class="sxs-lookup"><span data-stu-id="92cac-144">Cloud Services also provides monitoring.</span></span> <span data-ttu-id="92cac-145">Analogamente a Macchine virtuali di Azure, rileva un server fisico in errore e riavvia le VM in esecuzione su tale server in una nuova macchina.</span><span class="sxs-lookup"><span data-stu-id="92cac-145">Like Azure Virtual Machines, it detects a failed physical server and restarts the VMs that were running on that server on a new machine.</span></span> <span data-ttu-id="92cac-146">Servizi cloud consente inoltre di rilevare errori di macchine virtuali e applicazioni, non solo errori hardware.</span><span class="sxs-lookup"><span data-stu-id="92cac-146">But Cloud Services also detects failed VMs and applications, not just hardware failures.</span></span> <span data-ttu-id="92cac-147">A differenza di Macchine virtuali, dispone di un agente in ogni ruolo Web e di lavoro ed è quindi in grado di avviare nuove macchine virtuali e istanze di applicazione in caso di errori.</span><span class="sxs-lookup"><span data-stu-id="92cac-147">Unlike Virtual Machines, it has an agent inside each web and worker role, and so it's able to start new VMs and application instances when failures occur.</span></span>
