# <a name="set-up-your-media-services-development-environment"></a>Configurare l'ambiente di sviluppo di Servizi multimediali

I passaggi seguenti descrivono i prerequisiti richiesti per lo sviluppo con Servizi multimediali di Azure.

## <a name="create-a-media-services-account"></a>Creare un account di Servizi multimediali
Usare il portale di Azure, .NET SDK o l'API REST per creare un account di Servizi multimediali di Azure.

<a id="setup_dev_env"></a>

## <a name="set-up-the-development-environment"></a>Configurare l'ambiente di sviluppo

Per configurare l'ambiente di sviluppo .NET, vedere [questo](media-services-dotnet-how-to-use.md) argomento.  

Per configurare l'ambiente di sviluppo REST, vedere [questo](media-services-rest-how-to-use.md) argomento.  

<a id="connect"></a>

## <a name="connect-programmatically"></a>Connessione a livello di codice

Per connettersi all'API di Servizi multimediali di Azure, vedere [Accedere all'API di Servizi multimediali di Azure con l'autenticazione di Azure AD](media-services-use-aad-auth-to-access-ams-api.md).  

## <a name="next-steps"></a>Passaggi successivi

Gli esempi di codice sono disponibili nella raccolta di **Esempi di codice per Azure**: [Esempi di codice per Servizi multimediali di Azure](https://azure.microsoft.com/resources/samples/?service=media-services&sort=0).
